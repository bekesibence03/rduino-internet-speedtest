# rduino-internet-speedtest
Ez a projekt egy egyedi, hardveres hálózatfigyelő eszköz megépítését mutatja be, amely folyamatosan méri és vizuálisan megjeleníti az aktuális internetes letöltési sebességet (Mbps). A rendszer lelke egy Arduino UNO mikrokontroller és egy 4 digites, 7 szegmenses LED kijelző. A valós idejű hálózati adatok lekérését és soros porton (USB).
